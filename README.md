## About
This app was developed as part of [Hack For Peace](https://hackforpeace.net/).

## Installation
```
git clone git@github.com:Luca-Divit/Find.git
cd hack-for-eace
bundle
yarn install
bin/rails db:create
bin/dev
```
Open `localhost:3000` in your favourite browser
